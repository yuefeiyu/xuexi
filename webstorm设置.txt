webstrom不需要保存 ctrl+s
1.创建文件：
项目名称 右键-->new-->HTML File  新建html文件
                      css        css文件
                      js         js文件
2.修改字体大小
    file-->settings-->Editor-->Colors & fonts--> Scheme  Save As另存一份Scheme font-->size 调整字号
3.重命名（shift+F6）
文件右键 Refactor-->Rename -->Refactor
4.删除文件
Delete键
5.撤销
ctrl+z
6.软换行
file-->settings-->搜索Use Soft-->Gereral-->勾选Use Soft in  editor
7.修改文件默认编码
settings-->搜索Encoding  Utf-8
8.修改注册码
Help-->Register


常用的快捷键
ctrl+c  复制
ctrl+v  粘贴
ctrl+x  剪切
ctrl+z  撤销
Delete  删除
ctrl+d  复制一行
ctrl+f  搜索
